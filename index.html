Some ideas on Listal widgets

Motivation

Based on personal ideas and on some drafts by Pumpkinate, I'm trying to get some Listal data available for users to do random stuff, like putting widgets on their websites

My main driver is to be able to display my "Currently Reading Books" on my personal webpage. This is already working and, if no one else seems interested in something like this, it's where development will most likely end

Current implemented features are:

  • Get list of currently reading books by user (json and html ul) (try out json - html)
  • Get list of elements in custom list (json and html ul) (try out json - html)

Ideas for more features are:

  • Get list of wanted books/movies/music by user (json and html ul)
  • Get list of recently watched movies by user (json and html ul)
